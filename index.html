Flight Planner

Used Spark framework for Java and SQL Server to build a mock flight planner system for a multi-week project in my Data Management class. Powered by an Azure SQL Server, the program allows users to search for flights in an 87 Gb dataset of past flight data, and then book, pay for, and cancel flights. It also allows for safe concurrent transactions, multi-login user system, persistent user data, and SQL injection protection.
